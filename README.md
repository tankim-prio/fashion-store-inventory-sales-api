# Fashion Store Inventory & Sales Management API

A complete FastAPI backend portfolio project for managing fashion store inventory, customers, orders, payments, invoices, reports, AI Assistant, and Predictive Analytics.

---

## Main Features

- JWT authentication
- Admin and staff role system
- Category management
- Product management
- Product variant management
- Stock management
- Customer management
- Order management
- Payment management
- Invoice system
- Sales reports
- AI Assistant using local Ollama
- Predictive Analytics
- Demo seed data
- Automated tests

---

## Tech Stack

- FastAPI
- PostgreSQL
- SQLAlchemy
- Pydantic
- JWT Authentication
- Ollama AI Assistant
- Pytest
- HTML, CSS, JavaScript
