# Portfolio Summary

## Project Name

Fashion Store Inventory & Sales Management API

---

## Project Type

Backend API + Frontend Dashboard + AI Assistant + Predictive Analytics

---

## Problem Solved

Fashion stores need a system to manage:

- Products
- Stock
- Customers
- Orders
- Payments
- Invoices
- Reports
- Business decisions

This project solves those problems using a complete FastAPI backend and dashboard.
